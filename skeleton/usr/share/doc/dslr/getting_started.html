What is DSLR?

DSLR is a lightweight, single-user operating system.


What's special about DSLR?

DSLR ships with many classic, lightweight applications, adapted to run on modern hardware.


Where is the menu?

Right-click anywhere on the screen, press the DSLR button or ALT + F1.


How do I configure my wireless network card?

Go to the settings panel (through the Settings icon) and choose Wireless Wizard.


How do I enable remote access to DSLR?

Either enable Dropbear through the settings panel (for command-line access) or run x11vnc through the settings menu (for graphical access).


How do I install additional software?

Goto the settings panel and choose Package Manager.


What about security?

DSLR gives you the freedom to do whatever you want. However, this harms security, so critical system services or servers (e.g tftpd) are isolated from the rest of the system.

In addition, applications can be executed in a restricted environment (i.e so they cannot alter the system). If you mistrust an application or want to prevent it from reading your home directory, go to the menu and choose Run Restricted.
